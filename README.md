## Little Lost Robot

Little Lost Robot is a learning and passion project I am writing in Java, following the core architecture by RyiSnow on YouTube. 
It is a farming-themed RPG where players take on the role of a lost robot, Argo.
Players interact with obstacles, collectibles, and environments.

## Development Tools:

- Language: Java
- Graphics: Java AWT & Swing (JPanel, Graphics2D)
- Audio Library: Standard Java Sound API (javax.sound)
- IDE: IntelliJ IDEA
- Version Control: Git
- Art: Piskel App
